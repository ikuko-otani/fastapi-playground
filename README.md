# fastapi-playground

My FastAPI learning playground — the primary framework for my flagship project [`payment-ledger-api`](https://github.com/ikuko-otani).

## 🎯 Why this repo exists

FastAPI is listed as a core requirement (not nice-to-have) at several of my target employers: HelloFresh, Revolut, and others at EU scale-ups. This repo builds muscle memory for routing, dependency injection, Pydantic validation, and async endpoints before assembling them into the flagship.
